# Environment Variables

This document outlines the environment variables required for the ADD_TaskFlow-App, categorized by their usage in the backend and frontend.

---

## Backend Environment Variables (`backend/.env`)

These variables are used by the FastAPI backend.

| Variable Name             | Description                                          | Example Value                    |
|---------------------------|------------------------------------------------------|----------------------------------|
| `DATABASE_URL`            | Connection string for the PostgreSQL database.       | `postgresql://user:password@host:port/dbname` |
| `JWT_SECRET_KEY`          | Secret key for signing and verifying JWT tokens.     | `your-32-character-secret-key-here` |
| `JWT_ALGORITHM`           | Algorithm used for JWT signing.                      | `HS256`                          |
| `ACCESS_TOKEN_EXPIRE_MINUTES` | Expiration time for access tokens in minutes.    | `1440` (24 hours)                |
| `REFRESH_TOKEN_EXPIRE_DAYS` | Expiration time for refresh tokens in days.      | `7`                              |
| `ALLOWED_ORIGINS`         | Comma-separated list of allowed CORS origins.        | `http://localhost:3000,https://your.frontend.com` |

**Note:** `JWT_SECRET_KEY` should be a strong, randomly generated string. Use tools like `openssl rand -base64 32` to generate one.

---

## Frontend Environment Variables (`frontend/.env.local`)

These variables are used by the Next.js frontend. They should be prefixed with `NEXT_PUBLIC_` if accessed client-side.

| Variable Name             | Description                                          | Example Value                    |
|---------------------------|------------------------------------------------------|----------------------------------|
| `NEXTAUTH_SECRET`         | A random string used by NextAuth.js for hashing tokens, signing cookies, and cryptographic keys. | `your_nextauth_secret_here` (Generated by `openssl rand -base64 32`) |
| `NEXTAUTH_URL`            | The canonical URL of your frontend application.      | `http://localhost:3000`          |
| `NEXT_PUBLIC_API_BASE_URL`| The base URL of the FastAPI backend API.             | `http://localhost:8001`          |

**Note:** `NEXTAUTH_SECRET` should be a strong, randomly generated string. Use tools like `openssl rand -base64 32` to generate one.
